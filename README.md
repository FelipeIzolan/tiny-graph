## 🚀 Input Examples

- Inequality:
    ```
    x > 1
    y >= 2
    ```
- Linear Inequality:
  ```
  y > x
  y < 2x
  x >= -y + 2
  x <= 2y
  ```
- Linear Equation:
  ```
  y = x
  y = 2x
  x = -y + 2
  x = 2y
  ```
